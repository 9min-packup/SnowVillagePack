# BlueNether

![bluenether](./../images/bluenether.png)

ネザーが青くなるリソースパックです。ネザーラック系のブロック(ネザーレンガを除く)は青く、ソウル系のブロックは白くなり、氷と雪のような見た目になります。また、溶岩とマグマブロックは水色になります。涼しげなネザーを楽しんでください。 <br>
