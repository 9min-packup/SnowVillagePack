# SnowballAlchemy

![SnowballAlchemy](./../images/snowballalchemy.png)
![SnowballAlchemy](./../images/upgrade.gif)

SnowballAlchemy (雪玉錬金) は、雪玉を使用した、冒険を助ける多様なカスタムレシピを追加します。短期間だけ遊ぶワールドでも、雪玉さえあれば資源を充実させることができます。<br>
